# Network Intrusion Detection System (NIDS)

## Overview

The Network Intrusion Detection System (NIDS) is a cybersecurity project developed using Python, Scapy, Flask, SQLite, and Chart.js. The system monitors network traffic in real time, detects suspicious activities such as Port Scan and SYN Flood attacks, stores alerts in a database, and displays them through an interactive web dashboard.

---

## Features

- Real-time packet capture using Scapy
- Port Scan Detection
- SYN Flood Detection
- Alert logging in SQLite database
- Web-based monitoring dashboard using Flask
- Alert severity classification (HIGH / CRITICAL)
- Attack statistics visualization
- Interactive dashboard with attack counters
- Timestamped security alerts
- GitHub-ready project structure

## Detection Modules

### 1. Port Scan Detection

The system tracks the number of unique destination ports accessed by a source IP address.

If the threshold is exceeded:

```text
Attack Type : Port Scan
Severity    : HIGH
```

---

### 2. SYN Flood Detection

The system monitors TCP SYN packets generated by each source IP.

If excessive SYN packets are detected:

```text
Attack Type : SYN Flood
Severity    : CRITICAL
```

## Installation

### Clone Repository

```bash
git clone https://github.com/Dheeraj-cse29/Network-Intrusion-Detection-System.git
cd Network-Intrusion-Detection-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Project

### Step 1: Create Database

```bash
python database.py
```

### Step 2: Start Flask Dashboard

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

### Step 3: Start Packet Capture

Open a second terminal:

```bash
python packet_capture.py
```
